% Overview Section
\section*{Product Overview}

\textbf{Lithos} is an AI-powered research and diligence platform for the critical minerals mining industry. It automatically extracts structured data from 400--500 page technical reports, indexes 15,000+ mining projects worldwide, and provides instant AI-powered analysis that cuts analyst research time from \textbf{2 weeks to 2 minutes}.

\subsection*{Core Value Proposition}
\begin{itemize}
    \item \textbf{Automated Data Extraction:} LLM-powered pipeline extracts NPV, IRR, CAPEX, reserves, resources, and AISC from NI 43-101 and JORC technical reports
    \item \textbf{Global Coverage:} 15,000+ projects spanning gold, copper, lithium, nickel, rare earths, and other critical minerals
    \item \textbf{Instant Intelligence:} AI chat interface for risk analysis, project comparisons, and due diligence queries
    \item \textbf{Real-Time News Integration:} Automated tracking of permitting updates, production results, and commodity prices
\end{itemize}

% Core Features Section
\section*{Core Features}

\subsection*{1. Project Screener \& Database}
\textbf{Dense, Bloomberg-style table interface with advanced filtering and search}
\begin{itemize}
    \item \textbf{15,000+ Indexed Projects:} Global coverage of exploration through production-stage projects
    \item \textbf{Extracted Financial Metrics:} NPV, IRR, CAPEX, AISC, payback period, production rates
    \item \textbf{Resource \& Reserve Data:} Measured, indicated, inferred resources; proven and probable reserves
    \item \textbf{Advanced Multi-Filter System:}
    \begin{itemize}[leftmargin=2em]
        \item Filter by commodity (gold, copper, lithium, nickel, rare earths, etc.)
        \item Filter by country/jurisdiction and stage (exploration, feasibility, production)
        \item Range filters for NPV, IRR, CAPEX, AISC
        \item Risk level filtering (Low, Medium, High, Very High)
    \end{itemize}
    \item \textbf{Real-Time Global Search:} Instant search across project names, companies, locations
    \item \textbf{Column Management:} Show/hide columns, custom views, sortable headers
    \item \textbf{Bulk Actions:} Watchlist management, CSV export, multi-project comparison
    \item \textbf{Conditional Formatting:} Visual indicators for high IRR, low AISC, elevated risk
\end{itemize}

\subsection*{2. Project Detail Panel}
\textbf{Comprehensive single-project analysis with embedded documents and metrics}
\begin{itemize}
    \item \textbf{Key Metrics Dashboard:} NPV, IRR, CAPEX, payback, production forecasts
    \item \textbf{Technical Report Viewer:} Inline PDF viewer for NI 43-101 / JORC reports (400--500 pages)
    \item \textbf{Resource \& Reserve Breakdown:} Measured, indicated, inferred resources; P\&P reserves
    \item \textbf{Qualified Persons:} List of report authors with credentials and affiliations
    \item \textbf{Interactive Maps:} Geospatial visualization with project coordinates
    \item \textbf{Recent News \& Announcements:} Real-time permitting, production, and commodity updates
    \item \textbf{Risk Alerts:} Jurisdiction risk, commodity volatility, legal challenges
    \item \textbf{Similar Projects Recommendations:} AI-powered suggestions based on commodity, stage, location
    \item \textbf{Multi-Tab Interface:} Overview, Technical Report, AI Insights, News, Experts
\end{itemize}

\subsection*{3. AI-Powered Insights \& Chat}
\textbf{GPT-4-powered conversational interface for instant analysis}
\begin{itemize}
    \item \textbf{Project-Specific Questions:} "What are the main risks for Odyssey Project?"
    \item \textbf{Comparative Analysis:} "How does this compare to other gold mines in Ontario?"
    \item \textbf{Financial Deep Dives:} "What's driving the NPV? Break down the CAPEX."
    \item \textbf{Risk Assessment:} Instant analysis across geography, legal, commodity, and team dimensions
    \item \textbf{Document Grounding:} All answers cite technical reports, news, and regulatory filings
    \item \textbf{Context-Aware:} Remembers conversation history, references previous queries
    \item \textbf{Multi-Project Context:} Can analyze and compare multiple projects simultaneously
    \item \textbf{Export Conversations:} Save and share AI analysis for due diligence reports
\end{itemize}
